# Changelog

All notable changes to MoodLens will be documented in this file.

## [1.0.0] - 2024-07-05

### Added
- Complete emotion detection pipeline
- Streamlit web interface
- 6 emotion classes (Joy, Sadness, Anger, Fear, Love, Surprise)
- TF-IDF + Logistic Regression model
- Recommendation engine
- Word importance analysis
- Visualization utilities
- Comprehensive documentation

### Technical Details
- Python 3.12
- scikit-learn 1.3+
- Streamlit 1.28+
- Plotly for visualizations
- 90.4% accuracy on validation set

## [0.1.0] - 2024-07-01

### Added
- Initial project structure
- Data inspection script
- Configuration management
- Data loader with train/val/test splits
